# SI-04-3: Sprint 3 Test Script (Tenant Configuration & Provisioning)
**Project Name:** Project Mimir
**Sprint:** 3
**Feature:** Centralized Tenant Configuration and Provisioning Workflow

## คำแนะนำในการทดสอบ (Test Instructions)
1. รันระบบ Database (`docker-compose up -d`) เพื่อเตรียม MariaDB และ Qdrant
2. สังเกตและตรวจสอบระบบ Backend Initialization ให้ทำงานโดยไม่มีข้อผิดพลาด (Run `cargo run --bin monitor` หรือ `ro-ai-domain-game`)
3. รัน Frontend Dashboard โดยเข้าไปที่โฟลเดอร์ `ro-ai-dashboard` แล้วพิมพ์ `npm run dev`
4. ทำการทดสอบตามขั้นตอนด้านล่างสำหรับ Sprint 3 และ End-to-End Test Plan (TS-01 ถึง TS-05)
5. บันทึกผลในช่อง **"ผลการประเมิน"** (พิมพ์ `✅ Pass` หรือ `❌ Fail`) พร้อมแนบรายละเอียดหรือหมายเหตุในคอลัมน์สุดท้ายของตาราง

---
