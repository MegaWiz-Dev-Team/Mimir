# Security Policy

If you discover a security vulnerability in **Mimir**, please email **paripol@megawiz.co** instead of opening a public issue, pull request, or discussion.

We aim to acknowledge your report within **48 hours** and provide a fix or mitigation timeline within **7 days**. Please allow **up to 90 days** for coordinated disclosure before publishing details.

## Scope

This policy covers Mimir and its sub-crates (`ro-ai-bridge/*`, `ro-ai-dashboard/*`). For platform-wide issues that span multiple Asgard components, see the [Asgard SECURITY.md](https://github.com/MegaWiz-Dev-Team/Asgard/blob/main/SECURITY.md).

## Supported Versions

The latest commit on `main` is the only supported version.
